画像の中央値計算を中心に、各種のアルゴリズムを使用した、ブラウザ上で動作するカメラアプリです。

## なぜ人が消えるのか
人に限らず生物は基本的に動きます。それに対して背景となる風景はカメラを固定している限り動きません。

と言うことは、動いているものを消せば必然的に背景だけが残るのです。

動いているものを消す方法の一つに長時間露光がありますが、あれは長い撮影時間で撮れた画像の画像の平均値を使う方法で、人がいたところにはモヤがのこってしまいます。

バックエンドがGitHub pagesなのとソースコードは公開しているので画像が送信される心配はありません。安心してお使いください。

※作成はAIですがロジックの組み立てと検証は人力で行っています。
